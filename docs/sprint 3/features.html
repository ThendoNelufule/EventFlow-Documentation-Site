Announcement System

We implemented a real-time announcement feature:

  • Manager: Can post announcements instantly.
  • Staff: Receive and view announcements in real time without refreshing the page.

A reporting system was developed to handle incidents during events:

  • Staff: Can report incidents and attend to issues that match their role.
  • Guests: Can also report issues, which are then handled by staff members.
