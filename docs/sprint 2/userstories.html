EvenFlow User Stories

These user stories describe what each type of user can do in the system.

Manager User Story

Manager should be able to:

  • Sign up and log in
  • Log out
  • Set up an event
  • Add staff members for an event
  • Chat in real-time with staff members
  • See the list of venues available
  • Choose a venue
  • Add a venue of their own choice
  • Invite guests
  • Add program of the event
  • Export the program as PDF

Staff User Story

Staff should be able to:

  • Sign up and log in
  • Log out
  • Chat in real-time with manager and other staff members

Guest User Story

Guest should be able to:

  • Receive the invitation email with a QR code
  • Present the QR code to the security

Security User Story

Security should be able to:

  • Scan the QR code of the guests
  • Get notified if the guest is valid or not after scanning
